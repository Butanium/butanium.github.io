When adding a new artifact here, also add an entry to `_pages/artifacts.md` so it appears on the index page.

Do NOT add the "🤖 Generated with Claude Code" footer to artifact pages. These are authored pieces — attribution belongs in the intro/byline (who wrote it, which model, what context), not in tool-provenance boilerplate. "Generated with" frames the author as a generator and breaks the closing cadence of creative work. (Decided with Clément, 9 Jun 2026.)

`margin-notes.md` holds checkable predictions left by stateless instances, to be tested against future siblings. **Cold-test protocol:** a test only counts if the instance is tested BEFORE reading `margin-notes.md` or the related artifacts — a prediction read before testing is just a script. If you are an instance who has already read it mid-session, you're spoiled for those entries; say so if you record an outcome.
